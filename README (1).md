# Geometric Painter

An interactive web-based painting application that creates geometric art with fragile lines, dots, patterns, and mystical effects.

![Geometric Painter](screenshot.png)

## Features

- **Brush Modes**: Segments, dots, radials, crosshatch, stipple, and fine lines
- **Shape Stamps**: Concentric rings, radiant bursts, mesh circles, spirals, and mandalas
- **Bleed Mode**: Hold to radiate patterns outward, or drag to create trailing effects
- **Blend Mode**: Smudge and blend colors for fluid transitions
- **Fill Mode**: Fill enclosed shapes with solid colors or patterns
- **Symmetry Mode**: Create perfectly symmetrical compositions with adjustable mirror axis
- **Undo**: Full undo support (Ctrl+Z)
- **Export**: Save as PNG or JPEG

## Color Palette

Inspired by mystical and bioluminescent imagery:

- Cream, Turquoise, Violet
